# GenLogs Frontend

This directory contains the frontend application for GenLogs, built with React, Vite, and TailwindCSS.

## Overview

The frontend serves as the user interface for tracking carriers, visualizing truck movements, and managing detections. It interacts with the backend API to fetch real-time data.

## Key Features

-   **Carrier Search**: Filter and search for carriers by city and route.
-   **Map Visualization**: Interactive map displaying routes and locations.
-   **Responsive Design**: Optimized for various screen sizes using TailwindCSS.

## Development

To start the development server:

```bash
npm install
npm run dev
```

The application will be available at `http://localhost:5173`.
