# GenLogs Monorepo

This monorepo contains the source code for the GenLogs application, featuring a FastAPI backend and a React frontend.

## Structure

- **api/**: Backend services built with FastAPI and managed by `uv`.
- **frontend/**: Frontend application built with React and Vite.
- **infra/**: Infrastructure configuration (Docker, etc).

## Getting Started

### Prerequisites

- Node.js (for frontend)
- Python & uv (for backend)
- Docker (for containerized execution)

### Running Locally

See the README files in `api/` and `frontend/` for specific instructions.
